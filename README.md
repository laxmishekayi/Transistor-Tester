# Transistor Tester

## Overview
This project is an Arduino Nano-based Transistor Tester that identifies whether a transistor is NPN or PNP and determines its pin configuration (Base, Collector, and Emitter). The result is displayed on a 16×2 LCD.

## Features
- Detects NPN and PNP transistors
- Identifies Base, Collector, and Emitter pins
- Displays results on a 16×2 LCD
- Simple and easy to use

## Components Used
- Arduino Nano
- 16×2 LCD Display
- Push Button
- Resistors
- Breadboard
- Jumper Wires
- NPN/PNP Transistors

## Working Principle
When the push button is pressed, the Arduino applies different test voltages to the transistor terminals. Based on the measured voltages, it identifies the transistor type and its pin configuration. The result is displayed on the LCD.
